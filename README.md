# fuzzy-run
Just run it!

Run your npm scripts with the power of the fuzzy search,

## Install

```
npm install -g fuzzy-run
```

## Run

### List all package.json scripts

```
run
```

### Run a script 
You can run ```npm test``` with:

```
run test
```

or spell your script name incorrectly (that's the point of fuzzy-run)

```
run tst
```
